# Game Geo-Politics — GEOPOLITICAL COMMAND

## Stack
- Vue 3 + TypeScript + Vite 8
- Pinia state management
- Tailwind CSS 4 (@theme vars in src/styles/global.css)
- Chart.js + vue-chartjs
- Claude API (claude-sonnet-4-5) — player provides own key, direct browser calls
